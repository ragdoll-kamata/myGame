# 初期設定
    

    カード設定
        カード名:名前
        カードタイプ:カードタイプenum
        属性:属性enum
        カード説明:説明
        エネルギーコスト:属性enum, 数値(無属性を入れると多いほうのエネルギーを自動的に使う)
        発動条件:式  (あるなら)


    建物設定
        耐久値:耐久値
        発動タイミング:タイミングenum

    リソース
        画像:ファイルパス


# カード効果設定

    山札系
        表向きにする:枚数, 表向きにしたカード
        表向き選択:枚数,最小枚数, 最大枚数, 選択したカード
        山札を見る:枚数, 見たカード
        山札を見る選択:枚数,最小枚数, 最大枚数, 選択したカード
        山札下送り:送るカード(シャッフルされて送られる)
    
    墓地系
        墓地送り:送るカード
        裏向きで墓地送り:送るカード
        墓地選択:最小枚数, 最大枚数, 選択したカード
    
    手札系
        手札に加える:加えるカード
        保護:保護するカード
        手札選択:最小枚数, 最大枚数, 選択したカード

    フィールド系
        フィールド選択:最小枚数, 最大枚数, 選択したカード

    カード変数
        カード選択:カード, 最小枚数, 最大枚数, 選択したカード
        カードランダム選択:カード, 最小枚数, 最大枚数, 選択したカード

    カード変数フィルター(フィルター通したら条件にあってるカードをフィルタされたカードに分離)
        属性フィルター:属性, カード, フィルタされたカード
        タイプフィルター:タイプ, カード, フィルタされたカード

        //カード枚数:カード, 枚数

    関数呼び出し:関数名
