# 初期設定
    

    カード設定
        カード名:名前
        カードタイプ:カードタイプenum
        属性:属性enum
        カード説明:説明
        エネルギーコスト:属性enum, 数値(無属性を入れると多いほうのエネルギーを自動的に使う)
        発動条件:式  (あるなら)


    建物設定
        耐久値:耐久値
        耐久値減少条件:耐久値減少条件enum
        発動タイミング:発動タイミングenum

# カード効果設定

    山札系
        表向きにする:枚数, 表向きにしたカード
        表向き選択:枚数,最小枚数, 最大枚数, 選択したカード, 選択しなかったカード
        山札を見る:枚数, 見たカード
        山札を見る選択:枚数,最小枚数, 最大枚数, 選択したカード, 選択しなかったカード
        山札下送り:送るカード(シャッフルされて送られる)
    
    墓地系
        墓地送り:送るカード
        無公開墓地送り:送るカード
        墓地選択:最小枚数, 最大枚数, 選択したカード
    
    手札系
        手札に加える:加えるカード
        保護:保護するカード
        手札選択:最小枚数, 最大枚数, 選択したカード

    フィールド系
        フィールド選択:最小枚数, 最大枚数, 選択したカード

    カード変数
        カード選択:カード, 最小枚数, 最大枚数, 選択したカード
        カードランダム選択:カード, 最小枚数, 最大枚数, 選択したカード

    カード変数フィルター(条件にあってるカードをフィルタされたカードに分離)
        属性フィルター:属性enum, カード, フィルタされたカード
        タイプフィルター:タイプenum, カード, フィルタされたカード
        名前フィルター:名前, カード, フィルタされたカード　※


        ※名前の文字が入っていればフィルタされる。
            例文
                名前:光の 
                カード名:光の導き <-フィルタされる
                カード名:栄光 <-フィルタされない
                カード名:栄光の輝き <-フィルタされる
                カード名:光速のドロー <-フィルタされない


    関数呼び出し:関数名

# サンプル

    カードタイプ:儀式
    属性:光属性
    カード説明:説明
    エネルギーコスト:光属性, 2

}
効果{
    表向きにする:3,$カード
    属性フィルター:光属性, $カード, $フィルター
    手札に加える:$フィルター
}
